### Formalized

- Narrative entities as persistent state-bearing objects
- separation of canonical truth, knowledge, belief, memory, and interpretation
- consequence persistence as a provisional law
- momentum as accumulated consequential state change
- pressure as changing cost and availability of alternatives
- institutions as potentially independent narrative agents
- hope as credible valued continuation rather than reduced danger

### Architecture Implications

- world systems must advance beyond protagonist observation
- agents must act from situated knowledge
- relationships require directional history and memory
- failure should transform state rather than merely reset progress
- memory must preserve event provenance and consequence chains
- evaluation must remain multi-dimensional and uncertainty-aware
