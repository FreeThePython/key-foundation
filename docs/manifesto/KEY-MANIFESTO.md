## Purpose

KEY exists to create worlds with gravity: coherent realities with history, memory, consequence, and characters whose lives extend beyond the current scene.

KEY is not an infinite-content generator. It is a narrative intelligence engine built around meaningful choice, persistent truth, causal consequence, and stories that emerge from living worlds rather than predetermined scripts.
